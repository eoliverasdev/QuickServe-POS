# Carpeta de memòria

Aquesta carpeta conté els documents de la memòria del Projecte Final en format Markdown.

## Fitxers actuals

- `01_memoria_seccio_1.md`: Secció 1 redactada (amb alguns punts pendents de dades concretes).

## Proposta de format de treball

- Redactar en Markdown per velocitat i control de versions.
- Exportar a PDF quan estigui tancat el contingut.

## Opcions per passar a PDF

1. **Pandoc (recomanat):** qualitat molt bona i control de portada/estils.
2. **Typora/Obsidian/VSCode extensió:** exportació ràpida per una entrega acadèmica.
3. **Google Docs/Word:** copiar text final i maquetar si el centre demana un estil concret.

Quan tinguem les seccions 1, 2 i 3 tancades, podem generar un únic fitxer `memoria_final.md` i exportar-lo directament a PDF.
